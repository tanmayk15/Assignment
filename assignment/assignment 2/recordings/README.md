# Screen Recordings

Place screen recording videos here demonstrating:

1. **demo_quickstart.mp4** - Running the complete quickstart workflow
2. **demo_single_inspection.mp4** - Single image inspection process
3. **demo_batch_processing.mp4** - Batch processing workflow
4. **demo_results_review.mp4** - Reviewing outputs and reports

## Recording Checklist

- ✅ Show command execution in terminal
- ✅ Display generated sample images
- ✅ Show annotated output images side-by-side with originals
- ✅ Open and show JSON report contents
- ✅ Display summary visualization charts
- ✅ Demonstrate different defect types being detected

## Recommended Tools

- **Windows**: Xbox Game Bar (Win+G), OBS Studio
- **macOS**: QuickTime Screen Recording, OBS Studio
- **Linux**: SimpleScreenRecorder, OBS Studio

## Video Specifications

- Resolution: 1080p (1920x1080) or 720p (1280x720)
- Format: MP4 (H.264)
- Duration: 2-5 minutes per demo
- Include audio narration if possible
